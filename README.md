## Compass Susy Starter

This is a starter project template that Ruby Sass to compile Susy into CSS. This requires you to have Susy and Compass gems installed.

## Project Setup  

1. Install Susy and Compass 

Skip this step if you already have Susy and Compass gems installed. 

~~~
$ sudo gem install compass 
$ sudo gem install susy
~~~

2. Clone the repo 

~~~
git clone https://github.com/zellwk/compass-susy-starter.git
~~~

## Usage 

Run the following command

~~~
$ compass watch
~~~

This command tells Compass to watch files that are stated in config.rb and output them as instructed.

That's it!
